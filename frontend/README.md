# Frontend (React + Tailwind)

## Requisitos
- Node.js
- Backend Laravel corriendo en `http://127.0.0.1:8000`

## Instalar
```bash
npm install
```

## Ejecutar (dev)
```bash
npm run dev
```

Abre `http://127.0.0.1:5173`.

> Este frontend usa proxy de Vite: todo lo que empiece por `/api` se envía al backend (evita problemas de CORS).
